Introduction

Python is a high-level, interpreted, object-oriented programming language with dynamic semantics. Its high-level built-in data structures, combined with dynamic typing and dynamic binding, make it very attractive for Rapid Application Development.

Python supports modules and packages, which encourages program modularity and code reuse. The Python interpreter and the extensive standard library are available in source or binary form for most major platforms.

What you should already know

This guide assumes you have the following background:

  • Basic understanding of computer operations and text editors.
  • Some familiarity with programming concepts like loops and variables.
  • Knowledge of HTML or JavaScript is optional but can help with understanding syntax similarities and differences.
Python vs Other Languages

Python emphasizes readability and simplicity, often requiring fewer lines of code than other languages like C++ or Java. Unlike Java, Python doesn't use curly braces or semicolons to denote blocks — instead, it relies on indentation.

While JavaScript runs primarily in browsers, Python is more general-purpose and is used in web development, automation, AI, data science, and more.

Hello world

To write your first Python program, open any text editor, type the following, and save the file with a .py extension:

print("Hello, World!")

Run it using:

python your_file.py
Declaring Variables

In Python, you create variables simply by assigning a value. There’s no need to specify the type, as Python uses dynamic typing.

x = 10 # an integer name = "Alice" # a string price = 99.99 # a float is_valid = True # a boolean

Python automatically determines the type based on the value you assign.

Variable Scope

Variables created inside a function exist only inside that function:

def greet(): message = "Hi" print(message) greet() print(message) # ❌ NameError: message is not defined
Global Scope

Variables declared outside any function are global:

count = 0 def increment(): global count count += 1

Use global only when absolutely necessary — it makes code harder to maintain.

Constants

Python doesn’t have true constants, but by convention, constants are written in ALL_CAPS:

PI = 3.14159 MAX_USERS = 100

This is only a naming rule — Python will still let you reassign PI later, so discipline is on you.

Data Types

Python has several built-in types:

  • int – Integers (e.g., 10)
  • float – Decimal numbers (e.g., 3.14)
  • str – Strings or text (e.g., "hello")
  • bool – Boolean values (True, False)
  • list – Ordered, mutable collections
  • tuple – Ordered, immutable collections
  • dict – Key-value mappings
  • set – Unordered, unique items
  • NoneType – Represents “no value” (None)
